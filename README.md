### SaveMenu - "Shopify for restaurants"

Functional SaaS allowing restaurant owners to create online menues and process orders.

#### Features:
- web menu builder, desktop and mobile optimized
- food ordering flow with or without payments
- payment integrations with various processors
- merchant and customer email notifications
- merchant Telegram notifications
- merchant analytics digest (by email): orders, visits and referrals
- food icons library
- QR-code generation
- menu is installable as a PWA (Progressive Web App)
